## New Features

- 🧩 **Safe HTML Blocks** — Notes can now include sandboxed HTML blocks for small dashboards, demos, and custom visualizations while keeping execution isolated from the vault and app shell.
- ✅ **Todo Block Shortcut** — `Cmd+T` now toggles the current editor block between normal text and a todo/checklist block, making quick task capture feel native while writing.
- 🤖 **GitHub Copilot Agent Target** — Tolaria now supports GitHub Copilot CLI as another local AI agent option alongside the existing supported agent targets.
- 🧰 **Contribute Panel Refinements** — The contribution experience has been tightened so users can more easily understand how to support and participate in Tolaria.

## Improvements

- 📝 **More Reliable Markdown Editing** — Large-note parsing, Mermaid preservation, code block indentation, saved markdown escaping, and empty checklist handling are more robust.
- 🗂️ **Better Vault Organization Flows** — Nested folder creation, note retargeting, folder drag/drop, root moves, and path-style wikilinks with underscores received focused improvements.
- 🧠 **Smoother AI Workflows** — AI commit-message drafting, workspace conversation settings, quick prompt submission, and provider targeting were made more dependable.
- 🧱 **Cleaner Editor Commands** — Block type conversion, side-menu structure, current-block detection, and recovery around editor edge cases are steadier.

## Stability and Fixes

- Restored WebKit code block highlighting and tightened packaged PDF preview behavior.
- Hardened frontmatter metadata parsing, Git remote URL handling, Git sync on current upstream branches, Windows Tauri CLI startup, and WSL Git provider selection.
- Reduced noisy telemetry from benign stale/missing editor states and recovered editor race conditions more gracefully.
- Added or updated coverage for editor block resolution, todo shortcuts, Markdown serialization, localization, raw editing, at-link insertion, breadcrumb behavior, sheet formulas, and release-critical app paths.
